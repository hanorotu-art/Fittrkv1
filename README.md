# Iron Notes

A lightweight, phone-friendly workout tracker.

## Run it

For the simplest local test on a computer:

1. Unzip the folder.
2. Open a terminal in the folder.
3. Run:

   python -m http.server 8000

4. Open http://localhost:8000 in your browser.

On Android, host the folder using GitHub Pages, Netlify, or another static host, then use Chrome's **Add to Home screen** option.

## Included

- Workout routines
- Set, rep, and weight logging
- Workout history
- Basic PR/progress tracking
- Progressive-overload suggestions
- Backup export/import
- Offline support after first load

All data stays in the browser using localStorage.
